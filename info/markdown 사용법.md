# MarkDown 사용법
## 개요
- 1.장점
- 2.단점
- 3.사용법

## 1.장점
- 문법이 쉽고 간결
- 관리 용이성
- 별도의 도구 없이 작성 가능
- 다양한 형태로 변환 가능
- 텍스트(Text)로 저장되기 떄문에 용량이 적어 보관 용이  

## 2. 단점
- 표준이 없음
- 모든 HTML 마크업을 대신하지 못하는 한계

## 3. 사용법
## 3-1 제목(Header)
- #뒤에 띄어쓰기 넣기
- <h1>~<h6> 까지 표현 가능
```
# 제목1
## 제목2
### 제목3
#### 제목4
##### 제목5
###### 제목6
```
# 제목1

### 제목3
